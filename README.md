# URL Shortener Service

Author: Aakash Sachdev

## Prerequisites

Install Docker: https://docs.docker.com/get-docker/

## Steps to run

### Step 1: Download project

```bash
git clone https://github.com/aakashsachdev129/url-shortener.git
```

```bash
cd ./url-shortener
```

### Step 2: Run project
```bash
docker compose up
```
